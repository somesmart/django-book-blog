This is an experiment I've been working on to create database driven text adventure games. The idea being that you can structure the story and it's various outcomes, paths, etc. through an excel file or database table, and then use the same core "text adventure engine" to play the game. Once I get the core working, I will try to provide a way for others to create and upload their own games.
